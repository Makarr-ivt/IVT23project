
Для начала, ОДУ порядка $n \in \mathbb N$ выглядит так:
$$a_0(x) y^{(n)} + a_1(x)y^{(n-1)} + \ldots + a_n(x)y=0$$
Это уравнение можно записать кратко в виде
$$L[y] = 0$$
где
$$L[y] = a_0(x) y^{(n)} + a_1(x)y^{(n-1)} + \ldots + a_n(x)y$$
И $L[y]$ - это линейный дифференциальный оператор, соответствующий слинейному ОДУ.
Как слышно из названия, линейный дифференциальный оператор действительно линейный, и выполняются соответствующие линейные правила:
1. Умножение на скаляр:
$$L[Cy] \equiv CL[y]$$
$$\array{L[Cy] = a_0(x) (Cy)^{(n)} + a_1(x)(Cy)^{(n-1)} + \ldots + a_n(x)(Cy) = \\ =Ca_0(x) y^{(n)} + Ca_1(x)y^{(n-1)} + \ldots + Ca_n(x)y = \\ =C(a_0(x) y^{(n)} + a_1(x)y^{(n-1)} + \ldots + a_n(x)y) = CL[y]}$$
2. Линейность:
$$L[y_1 + y_2] = L[y_1] + L[y_2]$$
$$\array{L[y_1+y_2] = a_0(x) (y_1+y_2)^{(n)} + a_1(x)(y_1+y_2)^{(n-1)} + \ldots + a_n(x)(y_1+y_2) = \\ = \big[a_0(x) y_1^{(n)} + a_1(x)y_1^{(n-1)} + \ldots + a_n(x)y_1\big] + \ \\ \ + \big[a_0(x) y_2^{(n)} + a_1(x)y_2^{(n-1)} + \ldots + a_n(x)y_2\big] = L[y_1] + L[y_2]}$$

Соответственно получаем
$$L[\alpha y_1 + \beta y_2] = \alpha L[y_1] + \beta L[y_2]$$
Или вообще прям в общей форме
$$L\left[\sum_{i=1}^m c_iy_i\right] = \sum_{i=1}^m c_iL[y_i]$$
Из этих свойств вытекают теоремы, что если $y_1$ и $y_2$ - решения линейного однородного уравнения $L[y_1] = L[y_2] = 0$, то $Cy_1$ и $y_1+y_2$ тоже будут решениями. И как следствие, любая линейная комбинация решений линейного однородного уравнения тоже является решением этого уравнения (привет алгем!)
